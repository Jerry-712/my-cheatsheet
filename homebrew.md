# Homebrew 常用命令
| 命令 | 作用 |
  |------|------|
  | brew install 软件名  | 安装软件 |
  | brew uninstall 软件名| 卸载软件 |
  | brew update          | 更新 Homebrew |
  | brew upgrade         | 升级所有软件 |
  | brew list            | 列出已安装 |
  | brew search 关键词   | 搜索软件 |
  | brew info 软件名     | 查看软件信息 |
  | brew --cask          | 安装 GUI 应用 |
